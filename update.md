# 🔷 Data Prism

> Transform raw data into actionable insights directly inside Visual Studio Code.

Data Prism is a powerful VS Code extension designed for data scientists, machine learning engineers, researchers, students, and analysts who want to quickly understand datasets without leaving their development environment.

Analyze CSV and JSON files, detect data quality issues, explore statistics, and gain instant insights—all from within VS Code.

---

## ✨ Features

### 📂 Dataset Support

- CSV File Analysis
- JSON File Analysis
- Automatic Dataset Detection
- Large Dataset Handling
- Fast Parsing & Processing

---

## 📊 Dataset Overview

Get a complete summary of your dataset instantly.

### Includes

- Total Rows
- Total Columns
- Dataset Size
- Column Names
- Data Types
- Memory Usage

Example:

```text
Dataset Summary

Rows: 15,000
Columns: 24

Numerical Columns: 12
Categorical Columns: 10
Date Columns: 2
```

---

## 📈 Statistical Analysis

### Numerical Statistics

For every numerical column:

- Mean
- Median
- Mode
- Minimum Value
- Maximum Value
- Standard Deviation
- Variance
- Range

Example:

```text
Column: Salary

Mean: 54,321
Median: 51,000
Min: 20,000
Max: 180,000
```

---

### 📋 Categorical Analysis

Analyze text and category columns.

Features:

- Unique Values
- Most Common Values
- Frequency Distribution
- Category Counts

Example:

```text
Department

Unique Categories: 8

Top Values:
Engineering: 452
Sales: 311
Marketing: 187
```

## 🔍 Missing Value Detection

Quickly identify incomplete data.

Features:

- Missing Value Count
- Missing Value Percentage
- Column Completeness Analysis
- Dataset Completeness Score

Example:

| Column | Missing | Percentage |
|----------|----------|----------|
| Age | 120 | 12% |
| Salary | 35 | 3.5% |
| Department | 0 | 0% |

---

## 🧹 Duplicate Detection

Find duplicate records before training models or performing analysis.

Features:

- Duplicate Row Count
- Duplicate Percentage
- Duplicate Warnings
- Dataset Integrity Checks

Example:

```text
Duplicates Found: 42

Affected Rows: 0.8%

Recommendation:
Review and remove duplicate entries.
```

## 🚀 Commands

### Analyze Dataset

```text
Data Prism: Analyze Dataset
```

Runs a complete analysis of the selected dataset.

---

### Show Statistics

```text
Data Prism: Show Statistics
```

Displays detailed column statistics.

---

### Show Missing Values

```text
Data Prism: Missing Value Report
```

Displays missing value analysis.

---

### Generate Dataset Report

```text
Data Prism: Generate Report
```

Creates a full dataset profiling report.
